How to Use These Icons

Copy the SVG code from the code blocks above and replace the existing SVG content in your plastic-machining.liquid file for each respective capability block.

Icon Descriptions:

  • Planing: Represents a planer/shaper tool for surface finishing
  • Sawing/Blank Cutting: Shows a cutting blade with teeth for material separation
  • Milling: Depicts a milling cutter with multiple cutting edges
  • Turning: Shows a lathe operation with rotational arrows
  • Extruding: Represents material being pushed through a die
  • Welding: Shows welding equipment with flame/heat indication
